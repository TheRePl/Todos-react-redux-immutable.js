# Todo-react-redux-reselect-immutable
使用react,redux,reselect,react-redux,immutable.js等工具库，完成一个Todos案例。
## start

```youtrack
  $ npm install
  $ npm start
```
打开服务端：[http://127.0.0.1:3000](http://127.0.0.1:3000)
